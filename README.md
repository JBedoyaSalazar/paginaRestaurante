# 🍽️ Restaurante Web

![HTML](https://img.shields.io/badge/HTML-5-orange)
![SCSS](https://img.shields.io/badge/SCSS-Sass-pink)
![Status](https://img.shields.io/badge/status-en%20desarrollo-yellow)

Proyecto de desarrollo web enfocado en la creación de una **página para un restaurante**, diseñada para ofrecer una interfaz clara y atractiva donde los usuarios puedan conocer promociones, navegar por el sitio y realizar pedidos.

El objetivo es construir una base sólida para una plataforma digital moderna que permita mejorar la interacción entre el restaurante y sus clientes.

# 🚀 Tecnologías utilizadas

* **HTML5** → estructura del sitio web
* **SCSS (Sass)** → organización avanzada de estilos
* **CSS3** → diseño visual y animaciones
* **Flexbox** → distribución flexible de los elementos

# ✨ Funcionalidades actuales

El proyecto actualmente incluye:

### 🧭 Navegación principal

* Inicio
* Producto
* Promoción
* Nosotros
* Contacto

### 🏷️ Sección principal (Hero)

* Promoción destacada **(hasta 50% de descuento)**
* Descripción del servicio del restaurante
* Botón de llamada a la acción **"Ordena ya"**

### 🎨 Diseño visual

* Uso de **Flexbox** para organización del layout
* Botón interactivo con efecto **hover**
* Tipografía clásica para una presentación elegante
* Logo del restaurante en el encabezado

# 🧠 Conceptos implementados

Este proyecto aplica varios principios importantes del desarrollo frontend:

* **Separación de responsabilidades**

  * HTML → estructura
  * SCSS/CSS → estilos

* **Diseño modular con SCSS**

* **Layout flexible con Flexbox**

* **Interacciones visuales**

  * transform
  * hover
  * transiciones
